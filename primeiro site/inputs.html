Formulários e entrada de dados



Os formulários em html são representados pela tag input, ela possui diferentes entradas de dados: texto, número, e-mail, senha entre outros. Toda vez que criamos uma tag input ela cria uma caixa de entrada.



A tag input ela tem os parâmetros: type, para indicar um tipo de informação será inserida(text, password, checkbox, radio, number, e-mail), bem como o campo placeholder, é como se fosse uma dica para o usuário de qual informação precisa ser inserida ali, após clicar no campo esse texto desaparece.
