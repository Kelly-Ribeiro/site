Interação com links

Conceito de Links

Os links são utilizados para conectar páginas web. Para isso utilizamos a tag chamada de âncora: Os atributos que precisamos configurar na tag âncora:href: para inserirmos a url a ser carregada e target: para indicarmos se a nova página abrirá na mesma janela ou em janela diferente. target= _blanck (abre em outra guia) target= _self (abre na guia atual)

Quando não colocamos o atributo target o padrão self é aplicado, e a página é aberta na janela atual.
